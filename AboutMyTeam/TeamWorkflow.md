# Team Workflow
1.  ผู้นำทีมสร้าง Repo ด้วย GitHub Classroom
2.  สมาชิก Accept และ Clone repo ลงเครื่อง
3.  สมาชิกแต่ละคนสร้าง branch ของตัวเอง 
4.  แต่ละคนทำงานของตัวเองใน branch ของตนเองเท่านั้น
5.  เมื่อทำงานเสร็จ: add + commit + push
6.  เปิด Pull Request เพื่อขอ merge เข้าสู่ main
7.  ทีมช่วย review โค้ด
8.  หากมี conflict ให้แก้ไขร่วมกันในทีม
9.  เมื่อ merge เสร็จ → ทดสอบโปรแกรมรวม
10. ส่งงานขึ้น github เมื่อเสร็จสิ้น
